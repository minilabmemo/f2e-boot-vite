# f2e-boot-vite

這是一個練習專案，使用 vite 建立，加上 bootstrap 樣式設定。

## 練習項目

- bootstrap 樣式設定與客製
- 原始首頁：http://localhost:5173/
- 六角 API 商品購物管理 options API
  - 後臺 (需登入) http://localhost:5173/dashboard/products
    - 登入/操作商品上傳/訂單管理/優惠卷管理
  - 前台 http://localhost:5173/user/cart
    - 加入購物車/送出訂單
